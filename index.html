Eaglercraft Modding Tutorial

Welcome to the Eaglercraft modding tutorial! I'm Bruh_11181514, and together with George aka (Not an Orange), we'll guide you through creating your own mods for Eaglercraft.

What You’ll Need to Download

  • IntelliJ Community (Ultimate is optional but costs money)
  • IntelliJ Minecraft Development Addon
  • Blockbench (if you want to create custom logos or models)
  • Java 21 (As of 11/28/2024)
  • Gradle
  • Git
  • Optional:
    • WinRAR or 7zip
    • JDGUI
    • Node.js

Step 1: Set Up Your Workspace

Create a folder for your project to store the Eaglercraft source code and your mod files. For example, I’ve created a folder named MyEaglercraftMod.

Step 2: Download and Set Up IntelliJ IDEA

Download IntelliJ IDEA:

Visit IntelliJ IDEA Downloads and choose the Community Edition (free) or the Ultimate Edition (optional, paid).

Install IntelliJ IDEA:

Run the installer and follow the prompts to download IntelliJ on your computer. Make sure to enable the option to install the Java Development Kit (JDK) if prompted.

Open IntelliJ for the First Time:

Launch IntelliJ and let it initialize. Skip signing into a JetBrains account unless you’ve purchased the Ultimate Edition.

Step 3: Install the Minecraft Development Plugin

Open the Plugin Settings in IntelliJ:

Go to File > Settings > Plugins (Windows) or IntelliJ IDEA > Preferences > Plugins (macOS).

Search for the Plugin:

In the Marketplace tab, type "Minecraft Development" into the search bar.

Install the Plugin:

Click the Install button next to the Minecraft Development plugin.

Restart IntelliJ:

Once installed, restart IntelliJ to activate the plugin.

Step 4: Clone the Eaglercraft Repository

Install Git:

Download Git from Git Downloads and follow the installation prompts.

Open Command Prompt (or Terminal):

Navigate to the folder you created earlier (MyEaglercraftMod or your chosen name).

Clone the Repository:

Run the following command:

git clone https://github.com/Eaglercraft-Archive/EaglercraftX-1.8-workspace

Verify the Files:

After the command completes, your folder should contain the Eaglercraft source code files.

Step 5: Import the Project into IntelliJ

Open IntelliJ:

From the IntelliJ Welcome Screen, click Open.

Select the Folder:

Select your project folder (e.g., MyEaglercraftMod) and select it.

Wait for IntelliJ to Index the Project:

IntelliJ will analyze the files and set up the project. This might take a few minutes.

Set the JDK Version:

Go to File > Project Structure > SDKs. Add the correct JDK version (Java 8, 17, 21, or another version you’ve installed).

Step 6: Gradle Setup

Generate Gradle Files:

Click the “Gradle” tab on the right side of the screen and click the refresh icon.

Build the Project:

Once the Gradle files are synchronized, you can click on the green arrow to run the project.

Step 7: Adding a Custom Item

Create the Model for the Item

Navigate to \desktopRuntime\resources\assets\minecraft\models\item.

Add a new JSON file named {item_name}.json (e.g., netherite_sword.json). Inside this JSON file, define how your item should be rendered. For example:

{
  "parent": "item/generated",
  "textures": {
    "layer0": "minecraft:item/{item_name}"
  }
}

Replace {item_name} with your actual item name. This tells Minecraft how to display your item in the inventory.

Texture Preparation

Make sure you have a texture file for your item in the \desktopRuntime\resources\assets\minecraft\textures\items folder, and that it matches the name you used in the JSON file (e.g., netherite_sword.png).

Register the Item in Code

Navigate to src/game/java/net/minecraft/item/Item.java. Find the following line:

registerItem(431, "dark_oak_door", new ItemDoor(Blocks.dark_oak_door).setUnlocalizedName("doorDarkOak"));

After this line, add your custom item registration:

registerItem(432, "custom_item", new Item().setUnlocalizedName("custom_item").setCreativeTab(CreativeTabs.tabCombat));

Render the Item

Navigate to src/game/java/net/minecraft/client/renderer/entity/RenderItem.java. Find the line:

this.registerBlock(Blocks.dragon_egg, "dragon_egg");

After it, add:

this.registerItem(Items.custom_item, "custom_item");
